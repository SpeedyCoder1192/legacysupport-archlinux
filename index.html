About This Project

This project provides a user-friendly script designed to add legacy CPU and GPU support to Arch Linux installations. It's ideal for USB-based or portable setups intended to run on older or varied hardware.

The script detects older AMD and Intel hardware, installs the linux-lts kernel for improved compatibility, and offers an interactive setup process that lets users choose what to install. It also adds a convenient bootloader entry labeled Legacy Arch (linux-lts).
